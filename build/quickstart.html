Govstrap.io quickstart guide

1. Get Bootstrap

If you're already using Bootstrap to build your site then you're all set to style it with Govstrap.

If you don't have Bootstrap yet, download it from getbootstrap.com. Govstrap is built for Bootstrap version 4.0.0-beta2.

2. Add Govstrap

Download the Govstrap CSS file from GitHub.

Load it after Bootstrap's CSS, like this:


Example


<!-- Bootstrap core CSS -->
<link href="bootstrap.min.css" rel="stylesheet">

<!-- Govstrap styling -->
<link href="govstrap.css" rel="stylesheet">

3. You're done!

You can now use all of the Govstrap classes and elements in your site.
