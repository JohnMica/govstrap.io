To upload documents to be displayed on your web space:
 Create the folders www and www/htdocs
 Save your web site in /www/htdocs/

Your first page must be saved as one of the following:
 index.html index.htm

The server IS case sensitive.

Your website  should be available at http://www.<your domain name>
